LORIS' Imaging insertion pipeline is composed of several perl-based scripts and configuration files, which are housed in the [Loris-MRI git repo](https://github.com/aces/Loris-MRI).

To install, configure, and troubleshoot your Imaging pipeline setup -- 
Please follow the [Loris-MRI markdown documentation](https://github.com/aces/Loris-MRI/tree/master/docs/01-Introduction.md) (under `docs/`). 

> Note: Ensure you are consulting the docs from your release-specific branch

Imaging-related topics such as Anonymization, non-MRI modalities (such as EEG), BIDS-format compatibility, and pipeline script options are covered in these markdown docs. 

#### Post-processing in CBRAIN

LORIS is agnostic to external post-processing pipelines or analysis platforms, and can be connected to the CBRAIN processing portal.  Contact us for more information about CBRAIN links, and visit [CBRAIN.ca](http://cbrain.ca).
