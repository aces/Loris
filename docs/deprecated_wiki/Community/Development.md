If you would like to contribute to the LORIS codebase, please read our [[Contributing Guidelines|https://github.com/aces/Loris/blob/master/CONTRIBUTING.md]], fork our repository and send us a pull request.

Code is merged after a given pull request is reviewed using our [[code review checklist|Code-Review-Checklist]], has been approved by 2 reviewers, and is tagged [PassedManualTests](https://github.com/ aces/Loris/labels/Passed%20Manual%20Tests). 
The new [Pull Request template](https://github.com/aces/Loris/blob/master/.github/PULL_REQUEST_TEMPLATE.md) has guidelines for how to submit a contribution. 

If there are any issues with your pull request, please address them and assign it back to the original reviewer.

Please also see links to [Technical documentation](https://github.com/aces/Loris/wiki/Technical) such as the [Front-End Developer guide to LORIS](https://github.com/aces/Loris/blob/master/docs/React.README.md)
